# Glossary — Acme Employee Portal

> ⚠️ Fictional sample. Terms keyed by lowercase slug.

## sso
- **Term:** Single Sign-On
- **Definition:** A single authentication event granting access to multiple applications without re-entering credentials.
- **Synonyms / Abbreviations:** SSO
- **Related:** identity-provider, okta
- **Source:** SRC-MTG-2026-06-15

## identity-provider
- **Term:** Identity Provider
- **Definition:** The system that authenticates users and issues assertions trusted by other applications.
- **Synonyms / Abbreviations:** IdP
- **Related:** sso, okta
- **Source:** SRC-MTG-2026-06-15

## okta
- **Term:** Okta
- **Definition:** Acme's corporate identity provider, used to back SSO across internal applications.
- **Synonyms / Abbreviations:** —
- **Related:** identity-provider, sso
- **Source:** SRC-MTG-2026-06-15

## portal
- **Term:** Employee Portal
- **Definition:** The Acme internal web application for employee HR self-service (payslips, leave, profile). Maps to MOD-001.
- **Synonyms / Abbreviations:** the portal
- **Related:** —
- **Source:** SRC-MTG-2026-06-15

## contractor
- **Term:** Contractor
- **Definition:** A non-employee worker engaged by Acme. Portal access entitlement is currently undecided (see Q-002).
- **Synonyms / Abbreviations:** external worker
- **Related:** —
- **Source:** SRC-MTG-2026-06-15
